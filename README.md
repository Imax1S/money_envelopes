Turn saving money into an engaging daily ritual.

MoneyEnvelopes is a gamified savings tracker based on the digital envelope method. Set a financial goal, choose a timeline, and open a unique virtual envelope every day to reach your target effortlessly.

### Key Features
*   **Smart Distribution**: Choose from Equal, Linear Progression, or Random distribution modes.
*   **Gamification**: Earn achievements and track streaks to stay motivated.
*   **Flexible**: Support for multiple currencies and languages (English/Russian).
*   **Privacy First**: All data is stored locally in your browser. No registration required.

Built with React, TypeScript, and Tailwind CSS.
